# shop
这是一个简单通过Vue实现的外卖项目
因为没有后端,所有数据都是用mock模拟,所以会出现数据重复
项目在线预览地址: https://daysdash.github.io/Vue-eleme-APP/
使用手机打开,或者浏览器的移动端调试模式(需要刷新)  

已完成:
  主页,商家详情页,购物车



### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
